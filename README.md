\# Self-Healing Retail Pricing Platform



\## Overview

This project predicts product demand and optimizes pricing using machine learning.



\## Features

\- Demand prediction using Random Forest

\- Price optimization to maximize revenue

\- A/B testing logic for decision making

\- Flask API for real-time predictions



\## Tech Stack

\- Python

\- Scikit-learn

\- Flask

\- Pandas, NumPy



\## API Endpoint

POST /optimize



Example Input:

{

&#x20; "price": 100,

&#x20; "day": 2,

&#x20; "month": 5

}
