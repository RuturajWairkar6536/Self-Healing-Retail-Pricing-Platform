## 🎯 Quick Start (Choose One)

### ⚡ **FASTEST** - Mermaid Live Editor (2 min, 0 setup)
1. Go to → https://mermaid.live
2. Paste diagram content
3. Click "Download" → PNG/SVG

📖 **Full guide:** See `COMPLETE_DOWNLOAD_GUIDE.md`

### 💻 **DEVELOPER** - Mermaid CLI (5 min setup)
```bash
npm install -g mermaid-cli
mmdc -i diagrams/1-system-architecture.mmd -o diagram.png
```

📖 **Full guide:** See `README.md`

### 🐳 **NO SETUP** - Docker
```bash
docker run --rm -v $(pwd)/diagrams:/data aw3ly/mmdc \
  -i /data/1-system-architecture.mmd -o /data/diagram.png
```
